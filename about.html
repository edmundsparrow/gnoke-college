SCHOOL MANAGEMENT SYSTEM · v1.0

A complete, offline-first school management system for Nigerian secondary schools. Manage student enrolments, staff accounts, subject assignments, and academic results — all from a single admin panel, without ever connecting to a server.

Every record lives on this device in a local SQLite database. Your data never leaves the machine.

⚡ Offline-first 🔒 No server 📵 No account 🚫 No tracking
✨ What It Does
  • Register and manage Junior & Senior secondary students
  • Assign students to classes (JSS1–SS3, sections A–D)
  • Create Subject Teacher, Class Teacher, and Admin accounts
  • Record CAT scores, assignments, notes, coursework, and exams
  • View and filter per-class results by term and session
  • Export all databases as .db files for backup
  • Works completely offline — no internet required after first load
⚙️ Under the Hood
Database SQLite (sql.js / WASM)
Persistence IndexedDB
Network None required
Stack HTML · CSS · Vanilla JS
PWA Service Worker + manifest
Version v1.0
🗄️ Local Databases
students.db
StudentRegisterStorage · one table per class
staff.db

scores.db
